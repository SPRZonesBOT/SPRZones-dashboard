# 📊 SPRZONES Dashboard

> Real-time financial market monitoring dashboard with AI-powered recommendations, PEAD screener, Shariah compliance screener, and market alert bot.

## ✨ Features

### 📈 Market Data
- **Global & Indian Indices** — S&P 500, NASDAQ, NIFTY 50, and more
- **Cryptocurrency** — BTC/USDT with real-time price updateshttps://github.com/SPRZonesBOT/SPRZones-dashboard/tree/main
- **Forex & Commodities** — XAUUSD (Gold), XAGUSD (Silver), EUR/USD
- **Ticker Strip** — Real-time market overview

### 🤖 AI-Powered Tools
- **AI Recommendations** — Smart buy/sell signals for crypto, forex, and stocks
- **PEAD Screener** — Post-Earnings Announcement Drift analysis with surprise percentages
- **Shariah Technical Screener** — Compliance check (D/E ratio, non-halal revenue) + technical indicators (RSI)

### 🔔 Alert & Monitoring
- **Market Alert Bot** — Create real-time price alerts (Above/Below)
- **Recent Analyses** — Trading history with success rate tracking
- **Stock Categories** — Small/Medium/Large cap performance

### 📰 News & Updates
- **Real-Time News** — Financial market news with impact labels (High/Medium)
- **Economic Calendar** — Key events and data releases
